# 天体论 · 占星学交互课程

基于 Helena Avelar & Luís Ribeiro 著作《On the Heavenly Spheres》（2010）的前三章内容构建的交互式占星学入门课程。

## 课程内容

| 讲次 | 主题 | 内容 |
|------|------|------|
| 第一讲 | 占星学导论 | 起源、预测、四大分支、类型区分、六段历史时间轴 |
| 第二讲 | 占星宇宙观 | 日心vs地心、天球模型、第一性质、四元素、气质类型 |
| 第三讲 | 星盘入门 | 星盘结构、历史演变、核心要素、方盘vs圆盘 |

## 功能特性

- **TTS 语音朗读** — 每个内容区块配备中文语音合成
- **SVG 图解** — 内嵌矢量插图（天球模型、元素形成、星盘结构等）
- **交互元素** — 可展开时间轴、可点击天球层级、星盘演变画廊
- **知识测验** — 每讲4道选择题，即时反馈与详细解析
- **进度保存** — localStorage 自动持久化学习进度
- **浅色学术主题** — 温暖象牙白配色，适合长时间阅读学习

## 使用方式

直接在浏览器中打开 `lessons/index.html` 即可开始学习。
